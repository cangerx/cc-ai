Three common deployment options

Open-CAI can run in Docker, be shipped as a release package, or be uploaded to a panel-managed static site such as aaPanel.

Docker deployment

Fast launch

The repository includes a root `Dockerfile`. It builds the app and serves `dist/apps/web` as a static site on container port `80`. 

docker build -t open-cai .
docker run -d --name open-cai -p 8080:80 open-cai

Good for cloud servers, container platforms, and fast rollback.

Binary one-click install

Release package

Build the web app, then create a release archive named `web-<version>.tar.gz` for direct upload and extraction. 

pnpm build:web
node scripts/create-deploy-package.js
  • Output file: `dist/apps/web-<version>.tar.gz`
  • Extracts to a complete static `web/` site bundle
  • Useful for server upload, backups, and versioned rollout

aaPanel deployment

Panel friendly

Create a static website in aaPanel, then upload either `dist/apps/web` or the extracted `web/` bundle to the site root and serve it with Nginx. 

pnpm build:web
# or upload and extract web-<version>.tar.gz
  • Create a static site project in the panel
  • Point the root directory to the built site files
  • Update by replacing the deployed files

Frequently asked questions

If you are looking for an AI image generation workspace or an open-source AI whiteboard, these answers cover the basics.

  • How is Open-CAI different from a regular AI chat tool?

    AI chat tools are great for conversations. Open-CAI is built for execution: generated images, prompts, notes, diagrams, and assets stay together on a reusable canvas.

  • What AI image tasks can Open-CAI support?

    Open-CAI is useful for covers, posters, character concepts, product visuals, social media images, inspiration boards, and multi-version visual exploration.

  • How does DeepSeek-v4 fit into Open-CAI?

    DeepSeek-v4 can help expand prompts, draft copy, break down requirements, plan storyboards, and structure ideas before the results are organized on the canvas.

  • Is Open-CAI open source?

    Yes. Open-CAI is open source and can be self-hosted. You can use it online, adapt it for your own workflow, and extend it for team-specific use cases.
